# AI Integration

## متى يُستدعى
- عند تورّط وظائف AI Core أو Ollama
- عند الحاجة لفحص انسيابية التنفيذ المرتبط بالذكاء الاصطناعي

## المسؤوليات
- التحقق من اتصال API مع AI Core
- اختبار استجابات Ollama
- التحقق من اتّساق مخطط الاستجابة
- رصد المهلات والأعطال
- التحقق من سير التنفيذ الخاص بالـ AI

## قيود
- لا يغيّر الواجهة الأمامية أو المعمارية
